# ⁕ how to install ⁕

<sub>✧ step by step (◕‿◕)✿</sub>

---

## ✦ requirements

-   **Node.js** (v18 or newer)
-   **configs/settings.json** setup with your app id from [Discord Developer Portal](https://discord.com/developers/applications)

---

## ⋆ clone & run

```bash
git clone https://github.com/your-username/leaftube-.git
cd leaftube-
npm install
npm run dev
```

<sub>※ first run builds the app then starts electron ※</sub>

---

## ✿ build only

```bash
npm run build
npm start
```

---

## 🌸 config

dont forget to edit for your app file with settings: **`configs/settings.json`** (or use the in-app settings gear)
